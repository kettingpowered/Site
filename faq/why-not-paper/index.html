FAQ

What is Ketting?
Ketting is a Minecraft server hybrid, combing Forge, Bukkit and Spigot.
Who are the developers?
Our developers are listed on the team page.
What versions of Minecraft does Ketting support?
Check the downloads page for the latest supported versions.
How do I install Ketting?
When choosing a version from our downloads page, click 'Get it' and follow the provided instructions.
How do I install Ketting on Feather Client?
You can follow our provided YouTube tutorial.
You basically only need to add
-javaagent:server.jar
in the Launch Args field.
I found a bug, what do I do?
Go to our GitHub page and create an issue in the correct repository.
Is Ketting a continuation of Magma?
Ketting is not a continuation of Magma, but a project rewritten from scratch after Magma was discontinued.
Why did you use Spigot instead of Paper?
Due to the extra amount of patches we would need to apply (around 1500 on top of Spigot), we decided to use Spigot as a base. Paper also comes with some drastic changes to internal code that we didn't want to include.
